## GEMINI VEO GENERATION TIPS:

### Best Practices:
1. **Copy the entire prompt** including all details
2. **Generate one clip at a time** (8 seconds each)
3. **Download immediately** after generation
4. **Label files clearly**: `clip-01-frustrated.mp4`, `clip-02-phone-clock.mp4`, etc.
5. **Check quality** before moving to next clip
6. **Regenerate if needed** - Veo can produce variations

### Quality Settings:
- **Resolution:** 1920x1080 (Full HD)
- **Frame Rate:** 30fps (standard for YouTube)
- **Format:** MP4 (H.264 codec)
- **Aspect Ratio:** 16:9 landscape

### Common Issues & Solutions:
- **If motion is too fast:** Add "slow, smooth camera movement" to prompt
- **If quality is low:** Add "cinematic quality, high production value, professional"
- **If colors are off:** Specify exact color grading in prompt
- **If actions are unclear:** Break down actions into more specific steps

## VIDEO EDITING WORKFLOW:

### Step 1: Generate All Clips
- Generate 8 video clips using Gemini Veo
- Download and organize in numbered folders
- Review each clip for quality

### Step 2: Import to Editor
- Import all clips into video editor (Premiere Pro, Final Cut, DaVinci Resolve, CapCut)
- Arrange on timeline in sequence
- Trim any excess frames

### Step 3: Add Transitions
- Cross-dissolve between most clips (0.5 seconds)
- Fade to black for dramatic moments
- Match transition timing to voiceover

### Step 4: Add Audio
- Import voiceover audio from script
- Add background music (subtle, non-intrusive)
- Add sound effects (button clicks, success sounds)
- Mix audio levels properly

### Step 5: Color Grading
- Apply consistent color grade across all clips
- Match colors between clips for smooth flow
- Enhance contrast and saturation slightly
- Add subtle vignette for cinematic feel

### Step 6: Add Graphics
- Text overlays for key points
- Animated statistics and numbers
- Lower thirds for emphasis
- End screen with logo

### Step 7: Export
- Resolution: 1920x1080 (Full HD)
- Frame Rate: 30fps or 60fps
- Format: MP4 (H.264)
- Bitrate: 15-20 Mbps for high quality
- Audio: AAC, 320kbps
